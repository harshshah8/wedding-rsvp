# Harsh & Anubhuti Wedding RSVP

Single-page RSVP landing page for Harsh & Anubhuti.

## Run locally

```bash
npx serve -l 5173
```

Open `http://localhost:5173`.

## Deploy on Vercel

1. Push this repository to GitHub.
2. In Vercel, import the repository.
3. Framework preset: **Other**.
4. Build command: leave empty.
5. Output directory: root (static).
6. Deploy.
